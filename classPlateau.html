Description détaillée

Déclaration anticipée pour éviter les inclusions circulaires.

Représente le plateau de jeu.

Le plateau centralise toutes les données du jeu : villes, routes, pioche de cartes train, tickets destination et défausses. Il fournit également les méthodes de chargement depuis des fichiers CSV.

◆ chargerMapCsv()

void Plateau::chargerMapCsv ( )

Charge les routes depuis le fichier route.csv.

Voir également
chargerTicketCsv()

Le fichier doit contenir une ligne d'en-tête suivie de lignes au format :

villeA,villeB,longueur,couleur,couleur2

Si couleur2 vaut "Aucune", la route est simple ; sinon elle est double.

Note
Affiche une erreur sur stderr si le fichier est introuvable.
